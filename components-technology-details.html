Detailed intervention description

Software medical device that computes fractional flow reserve using standard coronary CT angiography images. Computer modeling program that provides a patient-specific analysis of coronary blood flow utilizing coronary computed tomography angiography (cCTA) scans and proprietary computer algorithms based on computational fluid dynamics; non-invasive, additional tool for clinicians who are considering the risks and benefits of invasive coronary procedures. Fractional flow reserve (FFR) is a pressure derived index of the maximal blood flow in a vessel with a stenosis expressed as a ratio of maximal blood flow in the same vessel in the absence of any stenosis. Image data, from a 64-slice or above cCTA scan, is sent securely using software installed on the local clinical imaging system to the HeartFlow central processing centre in Redwood City, California, USA. A HeartFlow case analyst uses CT images to create 3D computer models of the coronary arteries incorporating coronary flow characteristics. The results are presented in a report which is sent electronically to the referring clinician and includes 3D models of the coronary anatomy combined with functional information of the estimated FFR values (known as FFR-CT values). This information is designed to guide clinicians on management options for patients with coronary artery disease. HeartFlow FFR-CT is indicated for a non-invasive diagnosis of stable coronary artery disease. FFR-CT analysis is intended to support the functional evaluation of coronary artery disease.
